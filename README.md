# 🌱 e-Plant Shopping

A modern React-based e-commerce application for plant lovers. **e-Plant Shopping** allows users to browse a collection of plants, learn about the nursery, add items to a shopping cart, and manage their purchases through a clean and responsive user interface.

## 🚀 Live Overview

Paradise Nursery is a fictional online plant store designed to provide users with a seamless shopping experience for indoor and outdoor plants.

The application features:

* Beautiful landing page
* Product catalog browsing
* Shopping cart management
* State management using Redux Toolkit
* Responsive design
* Smooth navigation between pages

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/MubeenBhatti563/e-plantShopping.git
```

### Navigate to the Project

```bash
cd e-plantShopping
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

Application will be available at:

```bash
http://localhost:5173
```
